
# Student Result System (Python Tkinter GUI)

This project is a Python-based GUI application developed using the Tkinter library.
It calculates a student’s total marks, average, grade, and pass/fail status based
on marks entered for three subjects.

The application includes proper input validation and a user-friendly interface,
making it suitable for academic use and practical examinations.

---

## Features

- GUI-based application using Python Tkinter
- Student Name and Roll Number input
- Entry of marks for three subjects
- Total marks displayed out of 300
- Average marks calculation
- Grade calculation based on average
- Pass/Fail determination (Fail if any subject < 35)
- Input validation:
  - Only alphabets allowed for student name
  - Only numeric values allowed for marks
- Clear / Reset button
- Attractive and consistent color theme

---

## Grade Criteria

| Average Marks | Grade |
|--------------|-------|
| ≥ 90         | A+    |
| ≥ 80         | A     |
| ≥ 70         | B     |
| ≥ 60         | C     |
| ≥ 50         | D     |
| < 50         | E     |
| Any subject < 35 | F (Fail) |
